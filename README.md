## Getting Started

1. Open `index.html` in a web browser
2. Select a label by clicking on it
3. Use the toolbar to add text or images
4. Customize colors, fonts, and properties using the control panel
5. Drag elements within labels to reposition
6. Use the resize handle (small square) to resize elements
7. Print or save your design
